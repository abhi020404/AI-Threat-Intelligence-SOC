# AI-Threat-Intelligence-SOC
A real-time AI-driven Threat Intelligence Dashboard simulating a Security Operations Center (SOC) environment. This project collects and analyzes cyber threat data, classifies threats by severity (High / Medium / Low), and visualizes them in a dark-themed, hacker-style dashboard using Streamlit.

Key Features:

Generates realistic threat data (raw_attacks.json)

AI-style threat prioritization (analyzed_attacks.json)

Dark-themed SOC-style dashboard with metrics & tables

Visualizes High / Medium / Low risk threats for situational awareness

Tech Stack: Python, Streamlit, JSON, Pandas, AI-based risk scoring

Use Case: Learn AI-driven threat prioritization, SOC workflows, and build professional cybersecurity dashboards.
